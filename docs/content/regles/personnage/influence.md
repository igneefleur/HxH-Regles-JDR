<div class="cols" markdown>

L'influence décrit ce qu'une personne éprouve pour une autre, et la manière dont ces sentiments guident son comportement. Elle se mesure sur huit axes : chacun porte le nom d'un sentiment et va de son contraire le plus fort à sa forme la plus forte.

Ces règles sont optionnelles : le MJ s'en sert à sa guise.

Envers une même personne, chacun occupe une seule position par axe, mais les axes se combinent librement, même quand ils semblent se contredire : on peut aimer et se méfier, craindre et admirer.

<div class="keep" markdown>

### L'échelle des crans

Chaque axe prend une valeur parmi 0, ±1, ±3 et ±9 : 0 est le neutre, l'indifférence, l'état par défaut ; plus la valeur s'éloigne du neutre, plus le sentiment est fort. Ces crans ne sont que des indicateurs, ils donnent l'intensité du sentiment et guident le MJ dans les réactions de celui qui l'éprouve.

<div class="sepia-table crans-table" markdown>

| Cran | Ce qu'il produit |
|:---:|---|
| 0 | L'indifférence : aucun sentiment. |
| ±1 | Le sentiment teinte les réactions : préférences, humeurs, petits gestes. Celui qui l'éprouve reste maître de lui. |
| ±3 | Le sentiment engage : il fait agir. En positif, des risques limités, de vrais services ; en négatif, des refus, des évitements, des coups bas. |
| ±9 | Le sentiment gouverne : il domine la conduite, jusqu'au sacrifice de soi ou à la ruine de l'autre, et fausse le jugement. |

</div>

</div>

<div class="keep" markdown>

### Comportement

Le comportement d'une personne envers une autre se lit dans la somme de ses huit axes : elle lui est favorable au-dessus de +2, défavorable au-dessous de −2, neutre entre les deux.

<p class="formula">Comportement = somme des huit axes</p>

La table donne un aperçu de ses réactions selon la somme. Entre deux lignes, on retient toujours la plus proche de zéro : une somme de +8 relève de la ligne +6, une somme de −8 de la ligne −6.

<div class="sepia-table somme-table" markdown>

| Total | Comportement | Description |
|:---:|---|---|
| +48 | Inconditionnelle | Elle se sacrifierait pour lui. |
| +24 | Acquise | Elle l'aide même quand ça lui coûte. |
| +12 | Chaleureuse | Elle l'aide volontiers. |
| +6 | Amicale | Elle lui veut du bien. |
| +3 | Bien disposée | Elle le reçoit favorablement. |
| 0 | Neutre | Elle réagit selon les circonstances. |
| −3 | Froide | Elle le reçoit avec réserve. |
| −6 | Hostile | Elle ne lui veut pas de bien. |
| −12 | Malveillante | Elle lui nuit volontiers. |
| −24 | Acharnée | Elle lui nuit même quand ça lui coûte. |
| −48 | Implacable | Elle se sacrifierait pour l'abattre. |

</div>

Le comportement n'est lui aussi qu'un indicateur : il ne dicte pas les actes. Selon les circonstances, une personne hostile peut aider l'autre, et une personne amicale l'attaquer.

### Les huit axes

<div class="mcard" markdown data-cols="axe">
**Attirance** L'attirance est l'attrait que l'autre exerce sur le cœur, les sens ou l'esprit du personnage. Son contraire est la répulsion.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | passionné | Il brûle pour l'autre, qui occupe toutes ses pensées. |
| +3 | amoureux | Il l'aime et cherche à lui plaire. |
| +1 | séduit | L'autre lui plaît, son regard le cherche. |
| 0 | indifférent | L'autre ne lui fait ni chaud ni froid. |
| −1 | incommodé | Il le trouve déplaisant. |
| −3 | rebuté | Son contact lui répugne. |
| −9 | révulsé | Sa seule vue le dégoûte. |
</div>

<div class="mcard" markdown data-cols="axe">
**Amitié** L'amitié est l'attachement que le personnage porte à l'autre, le plaisir de sa compagnie comme le souci de son bien. Son contraire est l'inimitié.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | frère | Il le tient pour de la famille et veille sur lui. |
| +3 | ami | Sa compagnie lui est chère. |
| +1 | camarade | Il apprécie sa compagnie. |
| 0 | indifférent | Sa compagnie lui est égale. |
| −1 | lassé | Sa compagnie lui pèse. |
| −3 | brouillé | Il ne veut pas de lui dans la même pièce. |
| −9 | haineux | Il ne supporte rien de lui, pas même son nom. |
</div>

<div class="mcard" markdown data-cols="axe">
**Confiance** La confiance est le crédit que le personnage accorde à la parole et aux intentions de l'autre. Son contraire est la méfiance.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | crédule | Il croit tout ce que l'autre dit, sans jamais vérifier. |
| +3 | confiant | Il croit sa parole et s'y fie. |
| +1 | ouvert | Il l'écoute sans arrière-pensée. |
| 0 | indifférent | Il le croit ou non, selon les faits. |
| −1 | dubitatif | Il vérifie ce que l'autre avance. |
| −3 | méfiant | Il cherche le piège derrière chacun de ses mots. |
| −9 | incrédule | Il tient chacune de ses paroles pour un mensonge. |
</div>

<div class="mcard" markdown data-cols="axe">
**Estime** L'estime est la valeur que le personnage reconnaît à l'autre. Son contraire est le mépris.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | subjugué | Il le place au-dessus de tous. |
| +3 | admiratif | Il le cite en exemple. |
| +1 | respectueux | Il reconnaît sa valeur. |
| 0 | indifférent | Il ne le juge ni en bien ni en mal. |
| −1 | condescendant | Il le regarde de haut. |
| −3 | dédaigneux | Il le juge indigne de son temps. |
| −9 | méprisant | Il le tient pour moins que rien. |
</div>

<div class="mcard" markdown data-cols="axe">
**Gratitude** La gratitude est la dette que le personnage ressent envers l'autre, pour un bienfait reçu comme pour un tort qu'il lui a causé. Son contraire est la rancune.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | son obligé | Il lui doit la vie, ou tout comme. |
| +3 | redevable | Il lui doit beaucoup et veut s'acquitter. |
| +1 | reconnaissant | Il n'oublie pas ce qu'il lui doit. |
| 0 | indifférent | Ils ne se doivent rien. |
| −1 | froissé | Il garde un tort en travers de la gorge. |
| −3 | rancunier | Il attend son heure pour lui faire payer. |
| −9 | vindicatif | Faire payer le tort passe avant tout le reste. |
</div>

<div class="mcard" markdown data-cols="axe">
**Sérénité** La sérénité est le calme que le personnage trouve auprès de l'autre. Son contraire est la peur.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | inébranlable | À ses côtés, il se sent hors d'atteinte. |
| +3 | serein | Il se sent en sécurité près de lui. |
| +1 | apaisé | Sa proximité le détend. |
| 0 | indifférent | Sa proximité ne lui fait rien. |
| −1 | nerveux | Près de lui, il reste sur ses gardes. |
| −3 | craintif | Il évite de se trouver seul avec lui. |
| −9 | terrifié | Sa seule approche le glace. |
</div>

<div class="mcard" markdown data-cols="axe">
**Émulation** L'émulation est l'élan que les succès et les atouts de l'autre donnent au personnage. Son contraire est la jalousie.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | galvanisé | Chacun de ses succès le pousse à se surpasser. |
| +3 | inspiré | Ses succès le poussent à travailler pour l'égaler. |
| +1 | stimulé | Ses succès lui donnent envie de faire mieux. |
| 0 | indifférent | Ses succès ne lui font rien. |
| −1 | dépité | Ses succès lui laissent un goût amer. |
| −3 | jaloux | Ses succès le rongent. |
| −9 | hanté | Il ne vit plus que pour le surpasser. |
</div>

<div class="mcard" markdown data-cols="axe">
**Loyauté** La loyauté est la place que le personnage accorde à l'autorité de l'autre. Son contraire est la rébellion.

| Cran | Sentiment | Description |
|:---:|---|---|
| +9 | dévoué | Il le suivrait partout, sans discuter ses ordres. |
| +3 | fidèle | Il suit ses décisions, même sans les comprendre. |
| +1 | rallié | Il le suit de bonne grâce. |
| 0 | indifférent | Il suit ou non, selon son intérêt. |
| −1 | réticent | Il rechigne aux ordres et traîne des pieds. |
| −3 | frondeur | Il conteste ses décisions à voix haute. |
| −9 | rebelle | Il refuse ses ordres et se dresse contre lui. |
</div>

</div>
