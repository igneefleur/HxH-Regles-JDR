# Accueil

## Extension navigateur : la fiche HxH sur Roll20

Une extension Firefox affiche une fiche de personnage HxH à la place de la fiche
Roll20, et envoie ses jets dans le tchat. Les personnages se créent dans le
[créateur de personnage](personnage/index.md) ; l'extension les récupère et les
présente dans une partie Roll20, d'un simple bouton.

<p><a class="cl-link" href="download/hxh-roll20-firefox.xpi" download><strong>Télécharger l'extension (Firefox)</strong></a></p>

Installation temporaire (le temps de la session Firefox) :

1. ouvrir `about:debugging#/runtime/this-firefox` ;
2. « Charger un module complémentaire temporaire… » ;
3. choisir le fichier téléchargé.

Ensuite : ouvrir le [créateur de personnage](personnage/index.md) une fois (les
fiches se synchronisent), puis, dans une partie Roll20, cliquer le bouton
« Fiche HxH » en bas à droite et choisir le personnage à afficher.
